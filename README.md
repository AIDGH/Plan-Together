# Plan-Together, A Shared Real-Time Planner

A beautiful, synchronized daily, weekly, and monthly planner built specifically for two (Arad and Dorsa)! This Single Page Application (SPA) allows real-time task management with a custom-built Shamsi (Persian) calendar integration.

## ✨ Features
- **Real-Time Synchronization:** Powered by Firebase Firestore, every task added, checked, or deleted updates instantly on both screens without refreshing.
- **Custom Views:** Switch seamlessly between Daily, Weekly, and Monthly grid layouts.
- **Native Shamsi Calendar:** Features a fully custom-built Shamsi/Gregorian calendar engine written in Vanilla JavaScript, requiring zero external libraries.
- **Shared & Independent Tasks:** Assign tasks to specific users (Arad or Dorsa) with custom color-coded UI.
- **Floating Tasks & Backlogs:** Tasks that don't have a specific day roll over into weekly or monthly backlogs.
- **Sticky UI:** Fixed navigation and sidebars for a smooth scrolling experience on large grids.

## 🚀 Tech Stack
- **Frontend:** HTML5, CSS3 (CSS Grid & Flexbox), Vanilla JavaScript (ES6+ Modules)
- **Backend/Database:** Firebase (Firestore Realtime Database)
- **Typography:** Vazirmatn Font

## 💡 Architecture
The project avoids bulky frameworks and instead utilizes vanilla JavaScript modules and DOM manipulation to create a lightweight, blazing-fast real-time experience.
